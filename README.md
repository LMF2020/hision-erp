# 简单ERP管理系统
## 简要概述：
货架，货仓管理，简单的erp管理系统，很Cool，客户反映很好。
## 业务需求：
* 物料查询，
* 批量导入，
* 货架设置，
* 添加物料、物料上架、物料下架、货位之间导入物料
* 数据库配置下，自动建库表，开箱即用
